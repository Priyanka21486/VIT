Steps to be followed:


* It is a high compute code, so has to be run on an ADA account with min 2 GPUs for fast processing.
* After installing Conda environment, all the requirements given in requirements.txt file have to be downloaded.
* Python code has both the codes for classification and image retrieval.
* To run them, please comment classification to perform retrieval and vice versa.
* To run the code for retrieval with a fresh query, update the line with query to change the input.
